add_content.sh
==============

Usage:
------

.. productionlist::
   add_content.sh: add_content.sh path value [-h]

.. program:: add_content.sh

Description:
------------

ファイルへ値を追記します。

Arguments:
----------

.. option:: path

   値を追記するファイルのパス。

.. option:: value

   ファイルへ追記する値。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
