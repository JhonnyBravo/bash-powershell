clear_content.sh
================

Usage:
------

.. productionlist::
   clear_content.sh: clear_content.sh path [-h]

.. program:: clear_content.sh

Description:
------------

ファイルの内容を削除します。

Arguments
---------

.. option:: path

   内容を削除するファイルのパス。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
