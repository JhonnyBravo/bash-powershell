enable_vboxsf.sh
================

Usage:
------

.. productionlist::
   enable_vboxsf.sh: enable_vboxsf.sh user_name [-h]

.. program:: enable_vboxsf.sh

Description:
------------

Virtual Box のゲスト OS へ共有ディレクトリへのアクセス権を付与します。

Arguments:
----------

.. option:: user_name

   共有ディレクトリへのアクセスを許可するユーザの名前。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
