set_preferences.sh
==================

Usage:
------

.. productionlist::
   set_preferences.sh: set_preferences.sh path [-h]

.. program:: set_preferences.sh

Description:
------------

指定したファイルを .config/sublime-text-2/Packages/User/Preferences.sublime-settings
へコピーします。

Arguments:
----------

.. option:: path

   コピーするファイルのパス。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
