update_user_dirs.sh
===================

Usage:
------

.. productionlist::
   update_user_dirs.sh: update_user_dirs.sh [-e] [-j] [-h]

.. program:: update_user_dirs.sh

Description:
------------

ホームディレクトリ直下のディレクトリ名を英語化 / 日本語化します。
例) ~/ドキュメント -> ~/Documents

Options:
--------

.. option:: -e 

   ディレクトリ名を英語化します。

.. option:: -j 

   ディレクトリ名を日本語化します。

.. option:: -h 

   ヘルプを表示します。
