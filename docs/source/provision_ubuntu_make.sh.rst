provision_ubuntu_make.sh
========================

Usage:
------

.. productionlist::
   provision_ubuntu_make.sh: provision_ubuntu_make.sh [-i] [-u] [-h]

.. program:: provision_ubuntu_make.sh

Description:
------------

ubuntu-make をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   ubuntu-make をインストールします。

.. option:: -u 

   ubuntu-make をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
