set_key.sh
==========

Usage:
------

.. productionlist::
   set_key.sh: set_key.sh path key:value [-h]

.. program:: set_key.sh

Description:
------------

JSON ファイルのキーの値を更新します。

Arguments:
----------

.. option:: path

   JSON ファイルのパス。

.. option:: key:value

   キーの名前と新しい値。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
