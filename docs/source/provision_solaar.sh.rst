provision_solaar.sh
===================

Usage:
------

.. productionlist::
   provision_solaar.sh: provision_solaar.sh [-i] [-u] [-h]

.. program:: provision_solaar.sh

Description:
------------

solaar をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   solaar をインストールします。

.. option:: -u 

   solaar をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
