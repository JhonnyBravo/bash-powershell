remove_key.sh
=============

Usage:
------

.. productionlist::
   remove_key.sh: remove_key.sh path key [-h]

.. program:: remove_key.sh

Description:
------------

JSON ファイルからキーを削除します。

Arguments
---------

.. option:: path

   JSON ファイルのパス。

.. option:: key

   削除するキーの名前。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
