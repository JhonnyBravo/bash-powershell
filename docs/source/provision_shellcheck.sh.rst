provision_shellcheck.sh
=======================

Usage:
------

.. productionlist::
   provision_shellcheck.sh: provision_shellcheck.sh [-i] [-u] [-h]

.. program:: provision_shellcheck.sh

Description:
------------

shellcheck をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   shellcheck をインストールします。

.. option:: -u 

   shellcheck をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
