provision_vim.sh
================

Usage:
------

.. productionlist::
   provision_vim.sh: provision_vim.sh [-i] [-u] [-h]

.. program:: provision_vim.sh

Description:
------------

vim をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   vim をインストールします。

.. option:: -u 

   vim をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
