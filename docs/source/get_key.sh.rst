get_key.sh
==========

Usage:
------

.. productionlist::
   get_key.sh: get_key.sh path key [-h]

.. program:: get_key.sh

Description:
------------

JSON ファイルのキーの値を取得します。

Arguments:
----------

.. option:: path

   JSON ファイルのパス。

.. option:: key

   JSON のキー。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
