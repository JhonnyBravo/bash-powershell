set_pkg_ctrl.sh
===============

Usage:
------

.. productionlist::
   set_pkg_ctrl.sh: set_pkg_ctrl.sh path [-h]

.. program:: set_pkg_ctrl.sh

Description:
------------

指定したファイルを .config/sublime-text-2/Packages/User/Package Control.sublime-settings
へコピーします。

Arguments:
----------

.. option:: path

   コピーするファイルのパス。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
