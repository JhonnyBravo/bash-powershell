add_key.sh
==========

Usage:
------

.. productionlist::
   add_key.sh: add_key.sh path key:value [-h]

.. program:: add_key.sh

Description:
------------

JSON ファイルへ新しい key:value を追加します。

Arguments:
----------

.. option:: path

   JSON ファイルのパス。

.. option:: key:value

   JSON ファイルへ追加するキーと値。

Options:
--------

.. option:: -h 

   ヘルプを表示します。
