provision_build_tools.sh
========================

Usage:
------

.. productionlist::
   provision_build_tools.sh: provision_build_tools.sh [-i] [-u] [-h]

.. program:: provision_build_tools.sh

Description:
------------

gcc make をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   gcc make をインストールします。

.. option:: -u 

   gcc make をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
