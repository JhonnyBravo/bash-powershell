provision_neobundle.sh
======================

Usage:
------

.. productionlist::
   provision_neobundle.sh: provision_neobundle.sh [-i] [-u] [-h]

.. program:: provision_neobundle.sh

Description:
------------

neobundle をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   neobundle をインストールします。

.. option:: -u 

   neobundle をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
