provision_nodebrew.sh
=====================

Usage:
------

.. productionlist::
   provision_nodebrew.sh: provision_nodebrew.sh [-i] [-u] [-h]

.. program:: provision_nodebrew.sh

Description:
------------

nodebrew をインストール / アンインストールします。

Options:
--------

.. option:: -i 

   nodebrew をインストールします。

.. option:: -u 

   nodebrew をアンインストールします。

.. option:: -h 

   ヘルプを表示します。
